# Real Estate (Paged)


## Try this example on your local

```bash
npm create uhuu-starter@latest -- --example basic-paged
```

*Run Example with SSL*
```bash
# Watch changes and build uhuu.zip
npm run uhuu
# Run dev server with HTTPS
npm run dev:secure
# Build project and create uhuu.zip
npm run build
```
